# 🏥 Hospital Management System (GUI Version)

A **Hospital Management System** built using **Java Swing (Frontend GUI)** and **C++ (Backend Middleware)**.  
This project is designed to manage hospital operations like patient records, doctor information, appointments, and basic administrative tasks through a simple graphical interface.

---

## 🚀 Features

- 🧑‍⚕️ Add / View / Manage Patients
- 👨‍⚕️ Doctor records management
- 📅 Appointment scheduling system
- 💾 Data handling via C++ backend
- 🖥️ User-friendly Java Swing GUI
- ⚡ Fast and lightweight execution

---

## 🛠️ Tech Stack

- **Frontend:** Java Swing (GUI)
- **Backend:** C++
- **Communication:** File-based / system calls (if applicable)
- **IDE Used:** VS Code / IntelliJ IDEA (optional mention)
